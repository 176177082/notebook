  | >        | &gt;         |

* 元数据
  #+BEGIN_SRC html
    <meta charset="utf-8">
    <meta name="value" content="xxx">

  #+END_SRC

* 网站图标
  + 方式一 :: ~<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">~

  + 方式二 :: 将 ~favicon.ico~ 放在网站根目录， 浏览器会自动寻找它
