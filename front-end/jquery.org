* jQuery 语法
  + 基础语法： ~$(selector).action()~
  + 文档就绪事件 - 在 DOM 加载完成后才可以对 DOM 进行操作
    #+BEGIN_SRC javascript
      $(document).ready(function(){

        // 开始写 jQuery 代码...

      });

      // 简洁写法
      $(function(){

        // 开始写 jQuery 代码...

      });
    #+END_SRC
  + 选择器基于 ~CSS~ 选择器
