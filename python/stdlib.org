* 开发工具
  + doctest :: 通过 *文档字符串* 进行测试
    + [[https://docs.python.org/2/library/doctest.html][doctest — Test interactive Python examples]]

        if __name__ == "__main__":
            import doctest
            doctest.testmod()
      #+END_SRC

      参数 ~-v~ 列出详细的测试用例: ~python script.py -v~
