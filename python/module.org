* itertools
  [[http://wklken.me/posts/2013/08/20/python-extra-itertools.html][PYTHON-进阶-ITERTOOLS模块小结]]
  
  为高效循环创建迭代器

  + count(start, [step]) :: start, start+step, start+2*step, …

  + cycle(p) :: p0, p1, … plast, p0, p1, …

  + repeat(elem [,n]) :: elem, elem, elem, … endlessly or up to n times

  + chain(p, q, …) :: p0, p1, … plast, q0, q1, …

  + compress(data, selectors) :: (d[0] if s[0]), (d[1] if s[1]), …

  + groupby(iterable[, key]	) :: 按键值（v）分组的子迭代器

  + product(p, q, … [repeat=1]) :: product('ABCD', repeat=2) AA AB AC AD BA BB BC BD CA CB CC CD DA DB DC DD

       生成笛卡尔积， ~repeat~ 为重复次数。 如 ~[1,2,3], repeat=2~ ==> ~[1,2,3], [1,2,3]~

  + permutations(p[, r]) :: permutations('ABCD', 2)	 	AB AC AD BA BC BD CA CB CD DA DB DC

  + ... :: ...... 

  + [[https://docs.python.org/2.7/library/itertools.html][Python2]]
  + [[https://docs.python.org/3/library/itertools.html][Python3]]

* doctest
  文档链接： [[https://docs.python.org/2/library/doctest.html][doctest — Test interactive Python examples]]

  对文档字符串中的交互式文档进行测试：
  #+BEGIN_SRC python
    if __name__ == "__main__":
        import doctest
        doctest.testmod()
  #+END_SRC

  参数 ~-v~ 列出详细的测试用例: ~python script.py -v~
