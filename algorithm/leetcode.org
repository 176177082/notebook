* 矩形重叠
  矩形坐标以 ~[x1, y1, x2, y2]~ 的形式给出， 其中 ~(x1, y1)~ 为矩形 *左下角* 坐标，
  ~(x2, y2)~ 是矩形 *右上角* 坐标。

  判断给定的 *两个矩形* 是否重叠：
  #+BEGIN_SRC python
    class Solution:
        def isRectangleOverlap(self, rec1, rec2):
            """
            :type rec1: List[int]
            :type rec2: List[int]
            :rtype: bool
            """
            if rec1[2] > rec2[0] and rec2[2] > rec1[0]:
                if rec1[3] > rec2[1] and rec2[3] > rec1[1]:
                    return True
            return False
  #+END_SRC
