* 整数类型转换
  需要注意有符号数的自动类型提升问题，对于无符号数来算，类型提升时会在前面补 0，但是对于有符号数来说，前面补的是符号位。

  比如说：
  #+begin_src C
    char val = 0xe8;

    if (val == 0xe8) {  // 0xffffffe8 == 0x000000e8
      // ...
    }
  #+end_src

  位运算时也可能因此得到错误的结果：
  #+begin_src C
    char val = 0xe8;

    val ^ 0xff;  // => 0xffffffe8 ^ 0xff
  #+end_src

  对于无符号数来说，一般都能表现出正常的行为。
