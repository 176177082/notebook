* 数组相关
  初始化：
  #+BEGIN_SRC C
    int arr[] = {1, 2, 3};   // int arr[3] = {1, 2, 3}
    int arr[4] = {1, 2, 3};  // int arr[4] = {1, 2, 3, 0}
  #+END_SRC

  数组大小计算：
  #+BEGIN_SRC C
    sizeof(arr) / sizeof(arr[0]);
  #+END_SRC
