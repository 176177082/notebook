* 流操作
  + *中间操作* - 可连接起来的流操作 - 返回流
  + *终端操作* - 关闭流的操作 - 不返回流

  流使用的一般流程：
  #+BEGIN_EXAMPLE
    数据源 --> 中间操作 --> 终端操作
  #+END_EXAMPLE

* 注意事项
  + 和 ~Iterator~ 类似，流只能遍历一次。
