* 数据查询
  MySQL 中通过 LIMIT 子句（clause）限制结果数量：
  #+BEGIN_SRC sql
    SELECT * FROM tbl LIMIT 5;    -- 0 - 5
    SELECT * FROM tbl LIMIT 5, 5; -- 5 - 10
  #+END_SRC

  MySQL 支持正则表达式，用于替代 LIKE 子句：
  #+BEGIN_SRC sql
    SELECT * FROM tbl WHERE name REGEXP '.*';
  #+END_SRC
