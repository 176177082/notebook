  + 所有东西都需要包含在 class, interface, enum 中

* 布尔判断
  + Java 中 integer 和 boolean 是两种类型， 因此不能像这样：
    #+BEGIN_SRC java
      int x = 1;

      while (x) {}
    #+END_SRC
