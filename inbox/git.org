* 单独添加文件的 git 存储库
  #+begin_src bash
    $ rm .git/index                # 移除 index 文件
    $ git update-index --add file  # 添加文件
    $ git write-tree               # 构建树对象
    $ git commit-tree              # 构建提交对象
  #+end_src

  构建提交对象后，可以考虑为提交对象打一个 tag 进行保存。
