* 进制转化
  当 x = 2ⁿ, n = i + 4j(0 ≤ i ≤ 3).
  
  可以把 x 写成16进制数字 1(i=0), 4(i=2), 8(i=3) 开头， 后面紧跟 j 个 16 进制0.

  例： 2014 = 2**11, n = 11 = 3 + 4·2, hex = 0x800

* 字符
  十进制数字 x 的 ASCII 码正好是 0x3x
  
  a-z 为 0x61~0x7A

  a ^ a = 0

  a ^ b ^ a = b
  
* 掩码运算
  掩码 0xFF 获取一个二进制数据的最低8位数据， 其他位置为0

  如： 0x01234567 & 0xFF

* 不会零除
  num && 5 / num

* 移位
  左移补0

  逻辑右移补0 - 无符号数
  算术右移补最高有效位的值 - 有符号数
