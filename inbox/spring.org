* @DateTimeFormat
  Spring MVC 中的 @DateTimeFormat 是用来进行自动类型转换的，而带有 ~@RequstBody, @ResponseBody~ 注解的请求数据会通过 Json 解析器进行转换。

  因此，在实体类上应该使用 ~JsonFormat~ 而不是 ~DateFormat~.

  相关：HttpMessageConverter
