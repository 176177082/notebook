斜率大于 1 时：
\begin{align*}
  F(x + 0.5, y + 1) &= A(x + 0.5) + B(y + 1) + C \\
                    &= 0.5A + B                  \\
  
  F(x + 1.5, y + 2) &= A(x + 1.5) + B(y + 2) + C \\
                    &= 0.5A + B + A + B          \\

  F(x + 0.5, y + 2) &= A(x + 0.5) + B(y + 2) + C \\
                    &= 0.5A + B + B
\end{align*}

斜率小于等于 1 时：
\begin{align*}
  F(x + 1, y + 0.5) &= A(x + 1) + B(y + 0.5) + C \\
                    &= A + 0.5B                  \\
  
  F(x + 2, y + 1.5) &= A(x + 2) + B(y + 1.5) + C \\
                    &= A + 0.5B + A + B          \\

  F(x + 2, y + 0.5) &= A(x + 2) + B(y + 0.5) + C \\
                    &= A + 0.5B + A
\end{align*}

其中：
\begin{align*}
  A = -dy, B = dx
\end{align*}
