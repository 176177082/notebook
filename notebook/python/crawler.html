2 爬虫深度限制

通过限制爬取链接的最大深度和数量来避免无限爬取： 

def crawler():
    count, seen = 0, {seed_url: 0}
    while crawl_queue:
        url = crawl_queue.pop()
        if seen[url] < max_depth and count < max_count:
            for links in download(url):
                if link not in seen:
                    seen[link] = seen[url] + 1
                    crawl_queue.append(link)
                    count += 1
