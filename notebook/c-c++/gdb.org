* 简单指令
  |-------+--------------------------------------|
  | 指令  | 作用                                 |
  |-------+--------------------------------------|
  | n     | 单步调试，执行下一条语句，不进入函数 |
  | s     | 单步跟踪，执行下一条语句，进入函数   |
  | p var | 打印变量的值                         |
  |-------+--------------------------------------|
