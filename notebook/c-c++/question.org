* EOF 与无符号整数的比较
  保存读取到的字符的变量类型应该至少为 ~int~:
  #+BEGIN_SRC C
    while ((ch = fgetc(stdin)) != EOF)
  #+END_SRC

* 快速初始化数组
 #+BEGIN_SRC C
   bool met[MAXN * 2];
   // 初始化 met 数组为 0；
   memset(met, 0, sizeof(met));
 #+END_SRC

 不止是数组，结构体等占据一篇连续内存的对象，都可以通过类似的方式完成初始化。
