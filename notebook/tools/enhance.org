* Fping
  Fping 相较于 ping 命令可以同时 ping 多个目标 IP 地址，甚至可以将要测试的 IP 地址保存到文件中作为标准输入。

  #+begin_src bash
    $ fping < ip.txt
  #+end_src

  参考：
  + [[https://www.howtoing.com/ping-multiple-linux-hosts-using-fping][Fping - Linux 的高性能 Ping 工具]]
