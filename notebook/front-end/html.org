* 图片
  + ~img~ 标签显示 ~base64~ 编码格式的图片：
    #+BEGIN_SRC html
      <img src="data:image/png;base64,编码值">
    #+END_SRC

    参考 [[https://en.wikipedia.org/wiki/Data_URI_scheme#Syntax][Data URI scheme]]
