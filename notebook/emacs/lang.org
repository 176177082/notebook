* JavaScript
** 通过 Tern 进行自动补全
   可以通过配置文件 ~~/.tern-project~ 创建全局配置：
   #+BEGIN_SRC js
     {
       "libs": [
         "browser",
         "jquery"
       ]
     }
   #+END_SRC
