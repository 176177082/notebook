* 将 PriorityQueue 作为最小堆
  可以将 PriorityQueue 作为最小堆来使用：
  #+BEGIN_SRC java
    PriorityQueue minHeap = new PriorityQueue();
  #+END_SRC

  同时，只需要简单的变化就可以当做最大对来使用：
  #+BEGIN_SRC java
    Queue maxHeap = new PriorityQueue(Collections.reverseOrder());
  #+END_SRC
