* 带资源的 try 语句
  #+BEGIN_SRC java
    try (AutoCloseable a = new xxx();
         AutoCloseable b = new xxx()) {
      ...
    }
  #+END_SRC

  这一特性从 JDK 7 开始支持，实现 ~AutoCloseable~ 和 ~CloseAble~ 接口的对象都可以使用。
