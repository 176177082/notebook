#+TITLE:      Git 笔记

* github fork 的更新
** 给 fork 配置一个 remote
   =git remote add upstream sourse=

   remote 查看:
   =git remote -v=

** 同步 fork
   + 从上游仓库 fetch 分支和提交点, 传送到本地, 
     并会被存储在一个本地分支 upstream/master
     =git fetch upstream=
   + 切换到本地主分支:
     =git co master=
   + 把 upstream/master 分支合并到本地 master 上
     =git merge upstream/master=
* 克隆非 master 分支
  + 查看远程分支:
    =git branch -r #查看远程分支=
    或
    =git branch -a #查看所有分支=
  + 克隆分支:
    =git checkout 分支名=
    
* 大小写敏感
  #+BEGIN_EXAMPLE
  git config core.ignorecase false  # 设置大小写敏感
  # 修改文件名为小写
  git rm -r --cached .  # 从版本库删除大写文件
  git push ...
  #+END_EXAMPLE

* 本地分支提交
  1. git fetch origin
  2. git push
